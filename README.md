## Requirements

- macOS 14.0+
- Xcode 15.0+

## Contribution

```bash
git clone https://github.com/duongductrong/ClaudeShot.git
cd ClaudeShot
open ClaudeShot.xcodeproj
```

Build and run with ⌘R.

## Permissions

Screen Recording permission required. System Preferences > Privacy & Security > Screen Recording.
