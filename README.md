## Requirements

- macOS 13.0+
- Xcode 15.0+

## Contribution

```bash
git clone https://github.com/duongductrong/Snapzy.git
cd Snapzy
open Snapzy.xcodeproj
```

Build and run with ⌘R.

## Permissions

Screen Recording permission required. System Preferences > Privacy & Security > Screen Recording.
